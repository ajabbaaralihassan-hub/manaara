MANARA PLAZA RESORT - MENU IMAGES
===================================

Sida aad sawirrada ugu darsato:
1. Fur faylka CSV-ka ee "manara_image_list.csv"
2. Sawir kasta hel magaciisa sax (tusaale: beef_steak.jpeg)
3. Sawirka ku geli folderka saxda ah (tusaale: images/meat/)
4. Cabbirka sawirka: 400x300 pixel ama ka weyn

IMPORTANT / MUHIIM:
- Magacyada sawirrada noqon LAZIMIYAA sida CSV-ka ku qoran
- Format: jpeg ama PNG
- Cabbirka: 400x300 pixels (waa ugu fiican)
